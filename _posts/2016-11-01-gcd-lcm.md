–ù–µ –Ω–∞—Ä—É—à–∞—è –æ–±—â–Ω–æ—Å—Ç–∏, –ø–æ–ª–æ–∂–∏–º, —á—Ç–æ $$x > 0$$ –∏ $$y > 0$$, –≤ –ø—Ä–æ—Ç–∏–≤–Ω–æ–º —Å–ª—É—á–∞–µ –∑–∞–º–µ–Ω—è–µ–º –æ—Ç—Ä–∏—Ü–∞—Ç–µ–ª—å–Ω–æ–µ —á–∏—Å–ª–æ –Ω–∞ –æ–±—Ä–∞—Ç–Ω–æ–µ.

–ß–∏—Å–ª–æ $$k$$ –Ω–∞–∑—ã–≤–∞—é—Ç –æ–±—â–∏–º –¥–µ–ª–∏—Ç–µ–ª–µ–º, –µ—Å–ª–∏ $$k \mid x$$ (–≥–æ–≤–æ—Ä—è—Ç $$k$$ –¥–µ–ª–∏—Ç $$x$$) –∏ $$k \mid y$$.
–¢–æ–≥–¥–∞ $$\exists ! g$$ - –æ–±—â–∏–π –¥–µ–ª–∏—Ç–µ–ª—å —Ç–∞–∫–æ–π, —á—Ç–æ $$\forall k$$ $$k \mid g$$, —Ç–∞–∫–æ–π –æ–±—â–∏–π –¥–µ–ª–∏—Ç–µ–ª—å $$g$$ –Ω–∞–∑—ã–≤–∞—é—Ç –Ω–∞–∏–±–æ–ª—å—à–∏–º –æ–±—â–∏–º –¥–µ–ª–∏—Ç–µ–ª–µ–º, –∏ –æ–±–æ–∑–Ω–∞—á–∞—é—Ç $$g = –ù–û–î(x, y)$$.

## –ê–ª–≥–æ—Ä–∏—Ç–º –ï–≤–∫–ª–∏–¥–∞ –¥–ª—è —Ü–µ–ª—ã—Ö —á–∏—Å–µ–ª

–ú–æ–∂–Ω–æ –ø—Ä–µ–¥–ø–æ–ª–æ–∂–∏—Ç—å, —á—Ç–æ $$x > y$$, –≤ –ø—Ä–æ—Ç–∏–≤–Ω–æ–º —Å–ª—É—á–∞–µ –ø—Ä–æ—Å—Ç–æ –ø–æ–º–µ–Ω—è–µ–º $$x$$ –∏ $$y$$.
–¢–æ–≥–¥–∞ $$x$$ –º–æ–∂–Ω–æ –ø–æ–¥–µ–ª–∏—Ç—å —Å –æ—Å—Ç–∞—Ç–∫–æ–º –Ω–∞ $$y$$:

$$
x = y\cdot q_0 + r_1,
$$

$$q_0$$ –Ω–∞–∑—ã–≤–∞—é—Ç –Ω–µ–ø–æ–ª–Ω—ã–º —á–∞—Å—Ç–Ω—ã–º, –∞ $$r_1$$ - –æ—Å—Ç–∞—Ç–∫–æ–º –æ—Ç –¥–µ–ª–µ–Ω–∏—è, –ø—Ä–∏—á—ë–º $$0 \leq r_1 < y$$.
–î–∞–ª–µ–µ –ø–æ–¥–µ–ª–∏–º —Å –æ—Å—Ç–∞—Ç–∫–æ–º $$y$$ –Ω–∞ $$r_1$$:

$$
y = r_1\cdot q_1 + r_2,
$$

–ø—Ä–æ–¥–æ–ª–∂–∞—è —Ç–∞–∫ –¥–∞–ª–µ–µ –Ω–∞ –Ω–µ–∫–æ—Ç–æ—Ä–æ–º —à–∞–≥–µ $$n$$ –æ—Å—Ç–∞—Ç–æ–∫ $$r_n$$ –æ–∫–∞–∂–µ—Ç—Å—è —Ä–∞–≤–Ω—ã–º –Ω—É–ª—é, —Ç–∞–∫–∏–º –æ–±—Ä–∞–∑–æ–º –ø–æ–ª—É—á–∞–µ–º –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç—å:

$$
x > y > r_1 > r_2 > \dots > r_{n-1} > 0.
$$

–¢–æ–≥–¥–∞ $$–ù–û–î(x, y)$$ - –Ω–∞–∏–±–æ–ª—å—à–∏–π –æ–±—â–∏–π –¥–µ–ª–∏—Ç–µ–ª—å $$x$$ –∏ $$y$$ –±—É–¥–µ—Ç —Ä–∞–≤–µ–Ω $$r_{n-1}$$.

–§–æ—Ä–º–∞–ª—å–Ω–æ –∞–ª–≥–æ—Ä–∏—Ç–º –±—É–¥–µ—Ç –∏–º–µ—Ç—å –≤–∏–¥:

$$
\begin{equation*}
–ù–û–î(x, y) = 
 \begin{cases}
   x, &y = 0\\
   –ù–û–î(y, x\pmod{y}), &y > 0
 \end{cases}
\end{equation*}
$$

### –ö–æ—Ä—Ä–µ–∫—Ç–Ω–æ—Å—Ç—å

–ü—É—Å—Ç—å $$x = y\cdot q + r$$, —Ç–æ–≥–¥–∞ $$–ù–û–î(x, y) = –ù–û–î(y, r)$$.

$$–ù–û–î(r, 0) = r$$, $$\forall r > 0$$

### –†–µ–∞–ª–∏–∑–∞—Ü–∏—è

#### –†–µ–∫—É—Ä—Å–∏–≤–Ω–∞—è
{% highlight cpp %}
int GCD(const int x, const int y) {
    if (y == 0)
        return x;
    return GCD(y, x%y);
}
{% endhighlight %}

#### –ù–µ—Ä–µ–∫—É—Ä—Å–∏–≤–Ω–∞—è
{% highlight cpp %}
int GCD(const int x, const int y) {
    int r1(x), r2(y);
    while (r2 > 0) {
        r1 %= r2;
        swap(r1, r2);
    }
    return r1;
}
{% endhighlight %}

## –ù–û–ö

–ß–∏—Å–ª–æ $$m$$ –Ω–∞–∑—ã–≤–∞—é—Ç –æ–±—â–∏–º –∫—Ä–∞—Ç–Ω—ã–º, –µ—Å–ª–∏ $$x\mid m$$ –∏ $$y\mid m$$.

–¢–æ–≥–¥–∞ $$\exists ! n$$ - –æ–±—â–µ–µ –∫—Ä–∞—Ç–Ω–æ–µ —Ç–∞–∫–æ–µ, —á—Ç–æ $$\forall m$$ $$n \mid m$$, —Ç–∞–∫–æ–µ –æ–±—â–µ–µ –∫—Ä–∞—Ç–Ω–æ–µ $$n$$ –Ω–∞–∑—ã–≤–∞—é—Ç –Ω–∞–∏–º–µ–Ω—å—à–∏–º –æ–±—â–∏–º –∫—Ä–∞—Ç–Ω—ã–º, –∏ –æ–±–æ–∑–Ω–∞—á–∞—é—Ç $$n = –ù–û–ö(x, y)$$.

–í—ã—á–∏—Å–ª–µ–Ω–∏–µ –Ω–∞–∏–º–µ–Ω—å—à–µ–≥–æ –æ–±—â–µ–≥–æ –∫—Ä–∞—Ç–Ω–æ–≥–æ —Å–≤–æ–¥–∏—Ç—Å—è –∫ –≤—ã—á–∏—Å–ª–µ–Ω–∏—é –Ω–∞–∏–±–æ–ª—å—à–µ–≥–æ –æ–±—â–µ–≥–æ –¥–µ–ª–∏—Ç–µ–ª—è:

$$
